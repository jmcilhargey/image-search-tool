Image Search Tool

By Joe McIlhargey | GitHub

  • Entering a search string returns images from Google's API
  • Searching a URL in the address bar returns a JSON of the images with info
  • Database also stores recent search info and results can be paginated

Example Input

NASA Picture of the Day

https://image-search-tool.herokuapp.com/NASA Picture of the Day
